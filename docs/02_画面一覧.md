# 02 画面一覧

## 画面一覧

| 画面ID | 画面名 | パス | 実装状況 | 概要 |
| --- | --- | --- | --- | --- |
| SCR-000 | ログイン | `/login` | 実装済み | メールアドレス、パスワード入力とログイン操作を提供する |
| SCR-001 | ダッシュボード | `/dashboard` | 実装済み | 接続状態、件数サマリー、ステータス別KPI、最近更新された課題を表示 |
| SCR-002 | プロジェクト一覧 | `/projects` | 実装済み | プロジェクト一覧、検索、編集、削除、新規作成導線を表示 |
| SCR-003 | プロジェクト登録 | `/projects/new` | 実装済み | プロジェクトキー、名称、説明を登録 |
| SCR-004 | プロジェクト編集 | `/projects/:id/edit` | 実装済み | 既存プロジェクトを編集 |
| SCR-005 | 課題一覧 | `/issues` | 実装済み | 課題一覧、検索、フィルター、表示切替、詳細、編集、削除、新規作成導線を表示 |
| SCR-006 | 課題登録 | `/issues/new` | 実装済み | 課題を登録 |
| SCR-007 | 課題詳細 | `/issues/:id` | 実装済み | 課題の詳細項目を表示 |
| SCR-008 | 課題編集 | `/issues/:id/edit` | 実装済み | 既存課題を編集 |

`/` は `/login` へリダイレクトする。

## SCR-000 ログイン

### 表示内容

- メールアドレス入力
- パスワード入力
- ログインボタン

### 操作

- ログイン操作により `/dashboard` へ遷移する。

### 使用API

- なし

### 補足

- ログイン認証API連携、セッション管理、権限管理は未実装。
- 未実装項目はGitHub Issueとして起票し、実装管理する。

## 共通レイアウト

ログイン画面以外は、以下の共通レイアウトを使用する。

- 左固定サイドバー
  - ProjectFlowブランド
  - 新規プロジェクト導線
  - ダッシュボード、プロジェクト、課題メニュー
  - サンプルのプロジェクトナビゲーション
- 上部ヘッダー
  - ワークスペース表示
  - グローバル検索入力
  - 通知、ヘルプ、ユーザー表示
- メイン領域
  - ページ見出し
  - サマリー、タブ、フィルター、主要アクション
  - テーブル、ボード、タイムラインなどの作業領域

## SCR-001 ダッシュボード

### 表示内容

- Backend API接続状態
- プロジェクト数
- 課題数
- 未着手件数
- 対応中件数
- レビュー中件数
- 完了件数
- 最近更新された課題

### UI仕様

- KPIカードはステータスごとに背景色とアイコンを持つ。
- 最近更新された課題はテーブルで表示する。
- 最近更新された課題行をクリックすると課題詳細へ遷移する。

### 使用API

- `GET /api/health`
- `GET /api/dashboard`

## SCR-002 プロジェクト一覧

### 表示内容

- 表示件数
- プロジェクトID
- プロジェクトキー
- 名称
- 説明
- 更新日時
- 編集、削除アクション

### 操作

- キー、名称、説明による画面内検索
- 検索条件リセット
- 新規作成画面への遷移
- 編集画面への遷移
- 削除

### 使用API

- `GET /api/projects`
- `DELETE /api/projects/{id}`

## SCR-003 / SCR-004 プロジェクト登録・編集

### 入力項目

- プロジェクトキー
- プロジェクト名
- 説明

### 使用API

- 登録: `POST /api/projects`
- 編集: `GET /api/projects/{id}`, `PUT /api/projects/{id}`

## SCR-005 課題一覧

### 表示内容

- 表示件数
- 未完了件数
- 課題ID
- プロジェクトキー、プロジェクト名
- 件名
- ステータス
- 優先度
- 担当者
- 期限日
- 更新日時
- 編集、削除アクション

### 操作

- 課題名、プロジェクト、担当者、ステータス、優先度による画面内検索
- プロジェクトによるAPI絞り込み
- ステータスによるAPI絞り込み
- 検索条件と絞り込み条件のリセット
- 新規作成画面への遷移
- 詳細画面への遷移
- 編集画面への遷移
- 削除

### 表示切替

課題一覧画面では、同一データを以下の形式で切り替えて表示する。

- 一覧
- ボード
- ガント風タイムライン

現時点のボードとガント風タイムラインはフロントエンド上の表示切替であり、ドラッグ&ドロップ、日付編集、永続化、専用APIは未実装。

### 使用API

- `GET /api/issues`
- `GET /api/projects`
- `DELETE /api/issues/{id}`

## 表示ラベル

ステータスは以下のように表示する。

| 値 | 表示ラベル |
| --- | --- |
| `TODO` | 未着手 |
| `IN_PROGRESS` | 対応中 |
| `REVIEW` | レビュー中 |
| `DONE` | 完了 |

優先度は以下のように表示する。

| 値 | 表示ラベル |
| --- | --- |
| `LOW` | 低 |
| `MEDIUM` | 中 |
| `HIGH` | 高 |
